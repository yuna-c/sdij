[플레이어 설치] 플레이어가 계속 설치되는 오류

작성자 : SDIJ 관리자 작성일 : 2017-08-03 조회수 : 630

플레이어 설치를 진행 했음에도 정상적으로 설치가 되지 않는 경우 또는 관리자권한으로 윈도우가 로그인 되어 있지 않은 경 우, 툴바( TOOLBAR )및 백신 프로그램에 의한 설치 실패로 인한 경우로 설치가 반복될 수 있습니다.

우선 회원님의 PC에 로그인 시 관리자 권한 (Administrator)으로 로그인 되어 있는지 확인 해 주시기 바랍니다. ( 윈도우 시 작 > 제어판 > 사용자 계정 에서 확인하여 Administrator 계정 접속인지 확인하실 수 있으며 Administrator 계정이 아닌 경 우 해당 계정 접속하여 설치 시도 해 주시기 바랍니다.)
